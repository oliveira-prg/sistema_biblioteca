📚 Sistema de Gerenciamento de Biblioteca em C

Sistema de gerenciamento de biblioteca desenvolvido em linguagem C para controle de livros, usuários e empréstimos. O projeto foi construído utilizando estruturas (struct), enumerações (enum), modularização de funções e manipulação de vetores para simular um sistema real de biblioteca em ambiente de terminal.

O sistema permite cadastrar, consultar, atualizar e remover informações relacionadas ao acervo e aos usuários, além de realizar controle de empréstimos e devoluções.

Listagem apenas de livros disponíveis

🔄 Sistema de Empréstimos
